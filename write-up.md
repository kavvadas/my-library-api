# Μελλοντικές Βελτιώσεις και Επεκτάσεις του My Library API

Με επιπλέον χρόνο θα επικεντρωνόμουν κυρίως σε βελτιώσεις που θα έκαναν το project πιο επεκτάσιμο.

---
Είναι σημαντικό να αναφερθεί ότι έχω προσθέσει επιπλέον tests για καλύτερο test coverage σε edge cases, και health checks στο Docker setup.
---

Αρχικά, θα βελτίωνα το search functionality με περισσότερα φίλτρα και sorting options.
Για παράδειγμα, ο χρήστης θα μπορούσε να κάνει αναζήτηση μόνο σε διαθέσιμα βιβλία (status=available) ή
να ταξινομήσει τα βιβλία ανα χρονιά έκδοσης (sort=publication_year)

Επίσης, μια σημαντική επέκταση θα ήταν η προσθήκη ολοκληρωμένου user management και συστήματος δανεισμού βιβλίων. Αυτο θα απαιτούσε δημιουργία νέων database models και schemas για users και borrow records, καθώς και endpoints για authentication και διαχείριση χρηστών. 
Για παράδειγμα, κάθε χρήστης θα μπορούσε να έχει το δικό του ιστορικό δανεισμών, 
λίστα ενεργών checkouts και περιορισμούς στον αριθμό βιβλίων που μπορεί να δανειστεί.
Επιπλέον, θα μπορόυσαν να προστεθούν ρόλοι χρηστών (admin, member), με διαφορετικά permissions για διαχείριση βιβλίων και χρηστών.

Τέλος, αν υπήρχε επιπλέον χρόνος, θα πρόσθετα CI automation με GitHub Actions ώστε να εκτελούνται αυτόματα τα tests και οι βασικοί έλεγχοι ποιότητας σε κάθως pull/push request.
